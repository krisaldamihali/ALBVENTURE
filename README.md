# ALBVENTURE

ALBVENTURE is an interactive web project designed to promote tourism in Albania. The website showcases natural, cultural, and historical attractions through a visually engaging interactive map with location markers, representative images, and descriptive content.

## Key Features

- Interactive map with tourist points of interest across Albania
- Information cards with photos and descriptions for each destination
- Text-to-Speech and Speech-to-Text support for an accessible experience
- Focus on promoting local tourism through an intuitive web interface

## Project Purpose

The goal of ALBVENTURE is to raise awareness about Albania's tourism potential and help local and international visitors discover new places to visit.

## How It Works

The site uses standard web technologies to present an interactive map and destination details. Clicking on a point on the map displays the corresponding travel information, making it easy for users to explore each location.

## Repository Structure

- `index.html` - Main landing page for the website
- `explore.html` - Page for browsing and discovering destinations
- `game.html` - An interactive game or activity related to the project
- `map.html` - Map page with clickable tourist locations
- `css/` - Stylesheets for the website layout and design
- `js/` - JavaScript logic for interactivity and map behavior
- `img/` - Images used throughout the website

## Technologies Used

- HTML
- CSS
- JavaScript

## Usage

1. Open the project folder in a web browser.
2. Start with `index.html` to view the main site.
3. Explore the map and click on destination markers to view more details.

## Notes

This is a static web project that can be run locally without a backend server. For best results, open the HTML files in a modern browser such as Chrome, Firefox,or Edge.
